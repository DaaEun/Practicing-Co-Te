# 124나라의 숫자

124 나라가 있다. 124 나라에서는 10진법이 아닌 다음과 같은 자신들만의 규칙으로 수를 표현한다.

- 124 나라에는 자연수만 존재
- 124 나라에는 모든 수를 표현할 때 1, 2, 4만 사용

| 10진법 | 124 나라 |
|:----|:----|
|1|1|
|2|2|
|3|4|
|4|11|
|5|12|
|6|14|
|7|21|
|8|22|
|9|24|
|10|41|

#### Q. 자연수 n이 매개변수로 주어질 때, n을 124 나라에서 사용하는 숫자로 바꾼 값을 return 하도록 solution 함수를 완성하라

#### 제한사항
- n : 500,000,000 이하의 자연수
